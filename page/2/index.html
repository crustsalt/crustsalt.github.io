Java-容器学习

Java 容器学习

容器，就是一种Java中的可以容纳各种对象的数据结构，容器可以储存各种对象：例如String、int都可以，如果不确定要让容器接受哪一种对象，可以使用泛型来替代。但是如果使用泛型，如果往容器里存Integer、String两种甚至更多种不同的对象，虽然编译器不会报错，但是会报出有安全问题警告，所以尽量还是直接指定存储哪一种对象，而非使用泛型。
